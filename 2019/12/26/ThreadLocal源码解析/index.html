                // Â¶ÇÊûúÂÆå‰∫Ü‰πãÂêétableÂ§ßÂ∞èË∂ÖËøá‰∫Üthreshold - threshold / 4ÔºåÂàôËøõË°åÊâ©ÂÆπ2ÂÄç

            int len = tab.length;
            Entry e;

        private void resize() {
            Entry[] oldTab = table;
            int oldLen = oldTab.length;
            int newLen = oldLen * 2;

            int count = 0;

            for (Entry e : oldTab) {

        }

                    expungeStaleEntry(j);
            }
        }
